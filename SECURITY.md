# Security Policy

## Reporting a Vulnerability

We take the security of our systems seriously. If you believe you have found a security vulnerability in the SomosTécnicos (somostecnicos_FSM) project, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them to the project maintainers via email at:
`security@somostecnicos.com` (Replace with actual contact email)

You should receive a response within 48 hours. If for some reason you do not, please follow up via email to ensure we received your original message.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.x | :x:                |

## Disclosure Policy

We ask that you:

*   Do not disclose the vulnerability publicly until we have had a reasonable chance to correct it.
*   Do not potential exploit the vulnerability beyond what is needed to demonstrate the issue.
*   Do not attack our end users or their data.
