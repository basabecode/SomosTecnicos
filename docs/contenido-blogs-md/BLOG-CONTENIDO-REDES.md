# H1

WiFi no conecta: diagnóstico paso a paso

# INTRODUCCIÓN

"WiFi conectado, sin acceso a internet" es uno de los mensajes más frustrantes en tecnología doméstica. La causa puede estar en el router, en el dispositivo o en el proveedor — pero hay una forma de identificarlo rápidamente.

# CAUSAS COMUNES

- IP duplicada o conflicto de direcciones en la red local
- Contraseña guardada desactualizada en el dispositivo
- DNS mal configurado que impide resolver nombres de dominio
- Router con firmware desactualizado o con tabla ARP saturada
- Interferencia en banda 2.4 GHz por microondas o teléfonos inalámbricos

# CUÁNDO LLAMAR A UN TÉCNICO

Si todos los dispositivos del hogar tienen el mismo problema y el reinicio del router no lo resuelve, o si el problema apareció después de una tormenta eléctrica.
