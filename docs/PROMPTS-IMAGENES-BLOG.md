### Parámetros recomendados para Midjourney

```
[PROMPT] --ar 16:9 --style raw --v 6.1 --q 2 --no text, watermark, logo, cartoon, CGI glow
```

### Conversión a AVIF

Una vez generada la imagen en PNG o JPG de alta resolución (mínimo 1200×675 px), convertir a AVIF:

**Opción 1 — Squoosh (online, gratuito):**

1. Ir a [squoosh.app](https://squoosh.app)
2. Subir la imagen PNG/JPG
3. En el panel derecho, seleccionar `AVIF` como codec
4. Ajustar calidad a `70–80` (buen balance tamaño/calidad)
5. Descargar y renombrar según el nombre exacto indicado arriba

**Opción 2 — sharp (CLI Node.js, para procesamiento en lote):**

```bash
npx sharp-cli --input imagen.png --output imagen.avif --format avif --quality 75
```

**Opción 3 — ffmpeg:**

```bash
ffmpeg -i imagen.png -c:v libaom-av1 -crf 30 -b:v 0 imagen.avif
```
