## 🛠️ Comandos Esenciales (Desarrollo)

```bash
# 1. Configurar Base de Datos
pnpm db:push        # Sincronizar esquema
pnpm db:seed        # Poblar con datos de prueba

# 2. Iniciar Entorno
pnpm dev            # Servidor de desarrollo (localhost:3000)

# 3. Utilidades
pnpm db:studio      # Explorador de base de datos visual
```
